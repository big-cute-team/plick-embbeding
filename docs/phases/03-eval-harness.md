# Phase 03 — 정답 라벨 + 정량 평가 러너

## 목표

"눈 검증"을 정량화한다. 실데이터 셋에 정답 이슈 라벨을 붙이고, 어떤 실험
결과든 ARI·쌍 단위 정밀도·재현율(pairwise) 점수가 자동으로 나온다. 이 점수가
Phase 04 벤치마크의 채점 기준이 된다.

## 작업

- [ ] P03-T01 정답지 형식 정하기 — "이 기사는 이 이슈에 속한다"는 정답
      매핑(기사 id → 이슈 id)을 담을 파일 형식을 정하고, 그 파일을
      읽어오는 코드를 `data/labels/` 기준으로 만든다
- [ ] P03-T02 기사 56건에 정답 달기 — 예전 실험 기록에 남은 군집·오병합
      메모를 참고해 정답 초안을 만들고, 최종 확정은 개발자가 검수한다
- [ ] P03-T03 채점기 만들기 — 실험 결과를 정답과 비교해 점수를 내는 코드.
      군집이 정답과 얼마나 일치하는지(ARI), 기사 쌍 기준
      정밀도·재현율·F1, 그리고 잘못 묶이거나(오병합) 잘못 쪼개진(과분할)
      사례 목록까지 출력한다
- [ ] P03-T04 리포트에 점수 넣기 — 실험을 돌릴 때 정답지가 있으면 점수가
      리포트에 자동으로 포함되게 연결한다
- [ ] P03-T05 기준 점수 재기 — 지금까지의 잠정 구성(SEMANTIC_SIMILARITY,
      임계값 0.85)으로 점수를 측정해 앞으로 모든 실험의 비교
      기준(baseline)으로 기록한다
